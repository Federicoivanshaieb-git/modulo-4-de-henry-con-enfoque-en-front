# 🚀 Henry Bootcamp - Módulo 4: Especialización y Proyecto Final

Este repositorio contiene los proyectos y ejercicios desarrollados durante el cuarto y último módulo de la carrera de Full Stack Developer en Henry. Este módulo representa la culminación del aprendizaje técnico, integrando herramientas avanzadas para construir aplicaciones de nivel profesional.

## 📋 Descripción

En esta etapa final, me enfoqué en la optimización y escalabilidad de aplicaciones. Profundicé en el uso de TypeScript para un desarrollo más robusto, implementé estilos modernos con Tailwind CSS y exploré las capacidades de Next.js para mejorar el rendimiento y el SEO de mis proyectos.

---

## 🛠️ Tecnologías Utilizadas

*   **Next.js**: Framework de React para renderizado del lado del servidor (SSR) y generación estática.
*   **TypeScript**: Superconjunto de JavaScript que añade tipado estático para evitar errores en tiempo de ejecución.
*   **Tailwind CSS**: Framework de CSS basado en utilidades para un diseño rápido y responsivo.
*   **Autenticación y Pagos**: Integración de servicios externos (Auth0, Mercado Pago, Stripe).

---

## 🧠 Temas Aprendidos

### 1. Desarrollo con TypeScript
*   Tipado de variables, funciones y componentes de React.
*   Uso de Interfaces y Types para modelar datos de la API.
*   Configuración de entornos de desarrollo estrictos y seguros.

### 2. Next.js y Optimización
*   Estrategias de renderizado: **SSR**, **SSG** y **ISR**.
*   Optimización de imágenes y manejo de fuentes.
*   Enrutamiento basado en archivos (App Router) y Server Actions.

### 3. Herramientas de Estilo y UI
*   Maquetación avanzada y diseño responsivo con **Tailwind CSS**.
*   Implementación de componentes reutilizables y accesibles.
*   Manejo de estados complejos en formularios y validaciones.
